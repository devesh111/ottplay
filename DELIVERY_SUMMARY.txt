================================================================================
                         API ENDPOINTS IMPLEMENTED
================================================================================

1. Widget List API
   - Endpoint: GET /api/v4.5/web/widget/list
   - Purpose: Fetch featured content widgets
   - Hook: useWidgetList()

2. New Releases API
   - Endpoint: GET /api/v4.7/web/ranking
   - Purpose: Fetch newly released content
   - Hook: useNewReleases()

3. OTT Partners API
   - Endpoint: GET /api/v4.7/web/ranking
   - Purpose: Fetch streaming providers
   - Hook: useOTTPartners()

4. Provider Widget List API
   - Endpoint: GET /api/v4.5/web/widget/providerwidget/
   - Purpose: Fetch provider-specific widgets
   - Hook: useProviderWidgetList(providerId)

5. Provider Ranking API
   - Endpoint: GET /api/v4.6/web/ranking/providerwidgetranking
   - Purpose: Fetch provider-specific ranked content
   - Hook: useProviderRanking(providerId)

6. Movie Details API
   - Endpoint: GET /api/v4.6/mobile/movie/seo/url
   - Purpose: Fetch movie information
   - Hook: useMovieDetails(seoUrl)

7. Show Details API
   - Endpoint: GET /api/v4.6/web/show/seo/url
   - Purpose: Fetch TV show information
   - Hook: useShowDetails(seoUrl)

8. Episode Details API
   - Endpoint: GET /api/v4.6/web/episode/seo/url
   - Purpose: Fetch episode listings
   - Hook: useEpisodeDetails(seoUrl)

9. Live TV Details API
   - Endpoint: GET /api/v4.5/web/liveTV/details
   - Purpose: Fetch live TV channel info
   - Hook: useLiveTVDetails(seoUrl)

10. Sports Details API
    - Endpoint: GET /api/v4.5/web/tournaments/sports-details
    - Purpose: Fetch sports content
    - Hook: useSportsDetails(ottplayId)

11. Video Player API
    - Endpoint: GET /api/partner-service/v4.2/videoplayer
    - Purpose: Fetch streaming information
    - Hook: useVideoPlayer(params)

12. Search Autocomplete API
    - Endpoint: GET /api/search-service/v1.1/universal-autocomplete
    - Purpose: Fetch search suggestions
    - Hook: useSearchAutocomplete(query)

================================================================================
                         TROUBLESHOOTING
================================================================================

ISSUE: API Returns 401 Unauthorized
SOLUTION:
  - Check bearer token in lib/api/ottplay.js
  - Verify token hasn't expired
  - Ensure Authorization header is being sent

ISSUE: CORS Errors
SOLUTION:
  - Create Next.js API route as proxy
  - Or use environment variables for token

ISSUE: Images Not Loading
SOLUTION:
  - Check image URLs in API response
  - Verify image domains in next.config.js

ISSUE: Slow Performance
SOLUTION:
  - Enable caching in hooks
  - Implement React Query for server state
  - Use Next.js Image optimization
  - Enable compression in production
